# SimCo 市场报价分析器

**v1.27** · 一个 [Tampermonkey](https://www.tampermonkey.net/) 用户脚本，用于在 [SimCompanies](https://www.simcompanies.com/) 游戏中实时抓取并解析聊天室的买卖报价。

包含三个功能模块：
- **✈ 航天** — 航天产品（SOR/BFR/JUM/LUX/SEP/SAT）的专项报价汇总，含历史记录
- **🌐 全市场** — 任意产品的报价查询，通过关注列表自定义监控品类
- **📈 价格走势** — 航天产品历史价格折线图

## 安装

1. 浏览器安装 [Tampermonkey 扩展](https://www.tampermonkey.net/)
2. 点击 [Raw 链接](../../raw/main/simco_market_analyzer.user.js) 自动触发安装，或手动复制代码到 Tampermonkey 新建脚本
3. 刷新 simcompanies.com，右下角出现 **✈**、**🌐**、**📈** 三个按钮

---

## ✈ 航天模块

### 支持产品

| 代码 | 名称 | 别名 |
|------|------|------|
| `:re-91:` | SOR | sor, sors |
| `:re-94:` | BFR | bfr, bfrs |
| `:re-95:` | JUM | jum, jumbo, jumbos, jumbojet |
| `:re-96:` | LUX | lux, luxs, luxjet |
| `:re-97:` | SEP | sep, seps |
| `:re-99:` | SAT | sat, sats, satellite |

### 使用方法

1. 点击 **✈** 打开面板，有两个标签页：
   - **实时搜索** — 设置时间范围（默认 4 小时），点击 **🔍 搜索** 抓取航天聊天室 (X) 的最新报价
   - **历史记录** — 浏览本地缓存的历史报价，可按时间段筛选，支持设置截止时间
2. 结果按产品 / 质量等级展示买卖双方报价，点击价格可弹出原始消息详情

```
等级         BUY              SELL
Q4          88.5k×1          90k×1  92k×1
Q5          90k×1  93k×1    94k×1  94.5k×1
未明确等级   ×4               ×2
```

### 报价解析特性
- **直接调用游戏 API**，无需手动复制聊天记录
- `+/-Xk/Q` 逐级差价自动展开到 Q0–Q9
- 支持 `Q6/8 @935/965k` 等级与价格一一对应的斜线格式
- 支持 `Quality:/Price:` 结构化模板格式，包括单消息内多产品多价格的分块格式
- 支持省略 k 的写法：`@34.5`、`at 34.5` 等同于 `@34.5k`
- 跨行等级继承（上一行声明的等级可沿用到下一行产品）
- 租借消息自动过滤，同公司同价格去重

### 历史记录
- 每次搜索后自动将报价保存到 `localStorage`（按 Realm 分开存储）
- 最多保留 5000 条，最新 2000 条保存原始消息正文，更早的条目自动清空正文以节省空间
- 相同报价（同公司 + 同消息时间 + 同产品 + 同等级）自动去重，多次搜索不会重复存储

### 手动标记异常值
- 点击价格标签弹出的消息详情中，每条报价旁有 **⚠** 按钮，可手动标记为异常值
- 被标记的报价在价格列表中以橙色删除线显示
- 📈 价格走势图的"去异常"选项可过滤掉手动标记的异常报价
- 再次点击 **⚠ 异常** 可取消标记

## 🌐 全市场模块

### 使用方法

1. 点击 **🌐** 打开面板
2. 从下拉菜单选择要关注的物品（英文 / 中文显示），点击 **+ 关注** 加入列表
3. 每个物品可点击 Q0–Q12 标签过滤质量等级，**任意** 表示不限等级；关注列表持久化保存
4. 选择聊天室（Sales / Aerospace sales），设置时间范围，点击 **🔍 搜索**
5. 结果支持两种视图：
   - **汇总表** — 按产品 / 等级展示买卖价格，点击价格可查看原始消息弹窗
   - **原始消息** — 按时间从近到远列出所有匹配消息

### 支持的价格格式

| 类型 | 示例 |
|------|------|
| 直接报价 | `@34.5k`、`at 900k`、`$0.405` |
| 市场价百分比 | `mp-3%`、`-3% market`、`3% below mp` |
| 市场价差值 | `mp-100`、`-100 mp`、`100 below market` |

## 📈 价格走势模块

点击 **📈** 打开走势图面板，可按产品、品质、买卖方向、时间段筛选，数据来自本地历史记录缓存。勾选"去异常"可隐藏手动标记的异常报价。

---

## 通用特性

- **BUY/SELL 方向标签**：弹窗和原始消息列表均用绿色标注求购、红色标注出售
- **公司名可点击**，直达对应玩家主页
- **发送时间**：每条消息旁显示相对时间（如"3分钟前"）
- **已撤回消息**：标注"已撤回"并显示删除线
- 关闭面板后重新打开可恢复上次搜索结果（内存缓存）
- 面板可拖动，支持随时停止搜索
- **自动更新检测**：Tampermonkey 定期检查新版本；也可在面板底部"关于"区域手动点击"检查更新"
- 自带查询限速，约 1.2 页/秒，不滥用 API

## 注意

脚本运行在 simcompanies.com 同源下，使用浏览器已登录的 Cookie，无需额外授权。格式较为特殊的报价（如价格写在下一行、使用非常规符号等）可能无法识别，结果仅供参考。
